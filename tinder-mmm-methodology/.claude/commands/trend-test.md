하락 추세가 "실재"하는지 — 가정하지 말고 검정한다.

실행:
```bash
PYTHONPATH=src python -m mmm.cli trend
```

각 타깃에 대해 STL(데이터 기반 추세 형태), Mann-Kendall 4종(원시·자기상관보정·계절·탈계절),
ADF+KPSS(결정적 vs 확률보행), 1차차분 drift, 미디어 제거 후 잔차 MK를 본다.

판정 규칙:
- 가정 없는 검정이 전부 no-trend → 회귀의 `t` 계수는 **산물**. "구조적 하락" 주장 금지.
- 검정이 추세를 지지하고 ADF+KPSS가 추세-정상 → 진짜 결정적 하락(단 크기는 STL 기준으로 보고).

절대 회귀에 t를 넣고 그 계수만으로 추세를 단정하지 말 것(순환논증).
