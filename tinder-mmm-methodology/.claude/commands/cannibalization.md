하락이 카니발(유료가 오가닉 잠식)인지 오가닉(외생)인지 발라낸다.

실행:
```bash
PYTHONPATH=src python -m mmm.cli cannibalization
```

세 독립 검정의 삼각검증:
① 시간 선행성 — 램프 이전 저·평탄 지출 구간에서 이미 하락? (원인은 결과에 선행)
② 허위상관 — raw 음상관이 탈추세/1차차분에서 소멸·반전? (교란변수 = 시간)
③ 순증분 — net 유료 탄력성 > 0? (강한 카니발이면 ≤ 0)

2/3 이상 오가닉이면 "매체는 방어, 원인 아님". 결정적 증명은 지오 홀드아웃/오가닉-split이 필요함을 명시.
관측 상관만으로 인과(카니발)를 단정하지 말 것.
